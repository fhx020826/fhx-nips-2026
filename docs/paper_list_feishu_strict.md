说明：

- 这一版不再采用“最窄 strict”口径，而是按你的要求扩展为 `continuous VLN / VLN-CE 相关完整版`。
- 保留范围：
  - `continuous VLN / VLN-CE` 主任务论文
  - 直接服务于 `R2R-CE / RxR-CE / Habitat` 的 benchmark / dataset / challenge / 变体设定论文

  - UAV / long-horizon 等明确属于 VLN 问题空间且与连续导航强相关的扩展子线
- 仍然不纳入：
  - 纯离散 VLN 方法论文
  - 纯 survey
  - 与 continuous VLN 没有明确任务或 benchmark 绑定的泛 embodied navigation 工作
- 排序规则：按首次公开时间从新到旧排列；能核到 `arXiv v1` 的优先使用 `arXiv v1` 日期。
